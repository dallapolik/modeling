# Airport Runway Simulator
Симулятор взлётов/посадок с визуализацией полос.

## Возможности
- до 10 полос
- автоприсвоение полос
- цветовая индикация занятости
- добавление самолётов во время симуляции
- ускорение (F) / замедление (S)
- статистика по каждой полосе

## Формат flights.txt
```
start duration type name city
```
Пример:
```
0 5 takeoff Falcon21 Tokyo
3 7 landing TerraX -
```
Посадки всегда получают город Янташубе.

## Управление
- **F** – ускорить время
- **S** – замедлить
- **Статистика по полосе** – кнопка
- **Добавить самолёт** – кнопка

## Запуск
Проект Qt Widgets.
```
qmake
make
./AirportSimulator
```

## Логика полос
1. Если полоса свободна к моменту start → использовать её
2. Иначе создать новую, если <10
3. Иначе вывести ошибку
